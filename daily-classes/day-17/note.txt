Day:- 17 (CSS Grid)


Heading, paragraph, div - these all are block element
a element - inline element
Inline element don’t take full width, it takes that much width that needed.
Transform property don’t work in inline element.
With display: inline-block; we give an element both features of inline as well as block.

.box:nth-child(2n){} — Every 2nd
.box:nth-child(2n-1){} — 1,3,5
